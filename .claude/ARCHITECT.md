## 8. اكتشافات MHE الحرجة

> هذا القسم يوثّق ما تم اكتشافه عن MHE — مهم لفهم لماذا HITL يتصرف مختلفاً

### 8.1 MHE معطّل جزئياً [مؤكد من الكود]
**MHE gamma والسرعة مُتجاهَلة تماماً** في `_apply_mhe_correction()` (RocketMheNav.cpp سطر 474-543):
- التعليق الرسمي في الكود: "MHE gamma is fundamentally wrong (92° vs true 4°)"
- **فقط الموقع** (x, y, z) والرياح يُطبّقان
- **gyro bias مُصفّر**: `_gyro_bias.zero()` — مقبول لـ SITL, يجب تفعيله لطيران حقيقي
- **GPS velocity fusion**: alpha=0.8 — تصحيح قوي عند وصول بيانات GPS جديدة

### 8.2 مصدر حالة MPC فعلياً (في PX4/HITL)

| الحالة | المصدر | ملاحظة |
|--------|--------|--------|
| الموقع (x, y, h) | MHE → strapdown reset | يمر عبر MHE position correction |
| السرعة (V, gamma, chi) | Strapdown INS + GPS fusion | **MHE gamma مُتجاهَل!** |
| الاتجاه (phi, alpha, beta) | Strapdown quaternion + gyro | بدون bias correction حالياً |
| معدلات الدوران (p, q, r) | مباشرة من sensor_combined | gyro خام بدون bias |
| الزعانف (delta_*_act) | نموذج سيرفو (HITL) أو CAN (SITL) | HITL يتخطى CAN feedback |

### 8.3 MHE cost threshold
- الحد الطبيعي: 1e6
- **رُفع إلى 1e7 لـ HITL** لأن TCP latency يرفع التكاليف إلى 2-3M
- أول 10 حلول: حد 1e9 (warm start ضعيف)

**[مؤكد]**: بداية التباعد عند t≈2.0s.

**[فرضية]**: TCP latency (~20-50ms) يجعل MPC يرى حالة قديمة → يأمر بزعنفة خاطئة بفارق صغير → يتراكم مع الدفع (753N) إلى فرق pitch كبير.

**[فرضية]**: فروقات Python↔C++ الخمسة (D-001 إلى D-005) **تضاف** إلى مشكلة latency — ربما إصلاحها يقلل الفرق حتى لو لم يحل latency.

### 9.4 الإصلاحات المطبقة حالياً (غير كافية)
ثلاثة تعديلات في `RocketGncMpc.cpp` مشروطة بـ `_sys_hitl`:
1. gamma_ref bias: -2° أثناء coast
2. gamma_base: 250→180 أثناء boost
3. h_cruise_w مُضاعف

**النتيجة**: تحسن ~14% فقط (830→790م خطأ). **غير كافٍ**.

### 9.5 لماذا Python vs SITL مختلفان أيضاً؟
- Python pitch@5s: **4.8°** — SITL: **7.5°** (فرق +2.7°)
- لكن المدى متقارب: Python 2603م, SITL 2607م
- **[فرضية]**: Python يستخدم حالة مثالية. SITL يمر عبر strapdown INS + MHE → حالة مُقدّرة مختلفة قليلاً → MPC يتصرف مختلفاً → لكن يصحح نفسه. في HITL: نفس + latency يمنع التصحيح.
